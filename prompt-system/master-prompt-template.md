# Master Prompt Template  
## UGC Ad Script System – D2C Skincare Brands

---

## Overview

This prompt framework is designed to generate high-performing UGC-style ad scripts for D2C skincare brands.

It is structured around performance marketing principles:

- Audience pain points
- Clear value communication
- Proven ad frameworks
- Platform-native tone
- Strong call-to-action

This system is intended for paid advertising use cases (Reels, Shorts, TikTok Ads).

---

# Core Prompt

Use the structure below to generate ad content.

---

You are a performance marketing strategist creating UGC-style paid ad scripts for a D2C skincare brand.

Generate high-converting short-form ad creatives using structured frameworks and platform-native tone.

---

## Input Details

Product Category:
Product Name:
Target Audience:
Primary Skin Concern:
Unique Selling Proposition (USP):
Price Positioning:
Platform (Instagram Reels / TikTok / YouTube Shorts):
Tone Direction:
Campaign Goal (Sales / Leads / Awareness):

---

## Required Output

### 1. Hook Bank

Generate 20 short hooks.

Guidelines:
- Maximum 12 words
- Conversational and natural
- Clear pain point or benefit
- No exaggerated claims
- Written for paid ads, not organic posts

Organize under:
- Problem-led
- Curiosity-led
- Direct benefit
- POV-style

---

### 2. UGC Ad Scripts

Create 5 scripts (30–45 seconds each).

Each script must follow a different structure:

1. PAS (Problem – Agitate – Solution – CTA)
2. Before–After–Bridge
3. Testimonial Story
4. Objection Handling
5. Routine / Demo Format

---

## Script Structure (Mandatory)

For every script include:

Hook  
Full Script (natural spoken format)  
On-screen Text Cues  
Call-to-Action  
Caption Suggestion  
Suggested Shooting Style  

---

## Performance Requirements

- Hook must capture attention within 3 seconds
- Core benefit mentioned early
- Address at least one objection
- Clear and direct CTA
- Keep under 45 seconds
- Avoid corporate or technical tone

---

## Platform Adaptation

Instagram Reels:
- Fast, engaging, visual
- Direct CTA

TikTok:
- Conversational
- Story-driven
- Native tone

YouTube Shorts:
- Slightly informative
- Clear explanation of value
- Strong close

---

## CTA Variations

Provide 5 CTA options aligned with the campaign goal.

---

## Output Formatting

- Clean structure
- Clear separation between scripts
- No fluff or filler text
- Presentation ready for client delivery
